Vocabulaire

  • On appelle alphabet un ensemble fini de symboles. On notera nos alphabets \(\Sigma\).
  • Un mot est une suite finie de symboles \(\in \Sigma\).
  • La longueur du mot \(\omega\), est l'entier naturel \(l_{\omega} = |\omega|\).
  • Il existe un unique mot \(\epsilon\), tel que \(|\epsilon| = 0\).
  • L'ensemble des mots d'un alphabet \(\Sigma\) est \(\Sigma^*\)
  • Un langage \(L\) sur \(\Sigma\) est un ensemble de mots de \(\Sigma\).

Pour le reste de ce chapitre, nos mots seront implicitement des éléments de \(\Sigma^*\), sauf si explicite.

Les symboles n'ont pas besoin d'avoir une définition mathématique, on leur demande simplement de pouvoir les distinguer. Le symbole a est le symbole a, mais n'est pas le symbole b

Les mots étants des suites finies, nous pouvons ainsi les concaténer. Soit \(v = v_1v_2\cdots v_n\) et \(w = w_1w_2\cdots w_m\), alors \(v\cdot w = v_1v_2\cdots v_nw_1w_2\cdots w_m\). On notera \(vw\) pour exprimer \(v\cdot w\).

Le fait qu'un langage soit simplement un ensemble, nous gardons bien évidemment les opérations communes sur les ensembles.

Soit \(L_1, L_2\) deux langages sur \(\Sigma\), on a alors :

$$L_1 + L_2 = L_1 \cup L_2$$ $$L_1L_2 = L_1\times L_2 = \{vw~|~v \in L_1, w \in L_2\}$$ $$L_1 \setminus L_2 = \{w~|~w \in L_1, w \notin L_2\}$$ $$\overline{L_1} = \{w \in \Sigma^* | w \notin L_1\}$$

Il est important de noter que \((\Sigma^*, \cdot)\) est un monoïde, avec \(\epsilon\) comme élément neutre.

Soit \(L\) un langage, on a $$L^0 = \{\epsilon\}$$ $$L^{n+1} = LL^n$$ $$L^* = \bigcup_{i\geq0}{L^i}$$

Exemples

Soit \(\Sigma = \{a, b\}\), alors \(\Sigma^* = \{\epsilon, a, b, aa, ab, ba, bb, aaa, aab, \cdots\}\)

Soit \(L_p = \{w \in \Sigma^*~|~l_w~est~pair\}\) et \(L_i = \{w \in \Sigma^*~|~l_w~est~impair\}\), alors $$\{L_p, L_i\}~est~une~partition~de~\Sigma^*,~donc,~L_i + L_p = \Sigma^*$$ $$L_pL_i = L_iL_p = L_i$$ $$L_pL_p = L_p$$

Soit \(L_c\), le langage du langage de programmation C, alors

#include <stdio.h>
    
int main() { printf("Hello, World"); return 0; }

est un mot de \(L_c\).

\(\overline{L_c}\) est le langage des programmes syntaxiquement incorrects en C.

Ordre sur les mots

  • On dit que \(\alpha\) est un préfixe de \(\beta\) si il existe \(\omega\) tel que \(\beta = \alpha\omega\).
  • On dit que \(\alpha\) est un suffixe de \(\beta\) si il existe \(\omega\) tel que \(\beta = \omega\alpha\).
  • On dit que \(\alpha\) est un facteur de \(\beta\) si il existe \(\omega, \zeta\) tel que \(\beta = \omega\alpha\zeta\)

On definit \(\alpha \preceq_p \beta\) comme : \(\alpha\) est un préfixe de \(\beta\). Notre \(\preceq_p\) est une relation d'ordre, mais n'est pas totale.

Exemples

  • \(\alpha \preceq_p \alpha\beta\)
  • \(\alpha \preceq_p \alpha\), le préfixe est \(\epsilon\)
  • \(\epsilon \preceq_p \alpha\)

Cependant, avec \(\preceq_p\) il nous est impossible d'ordonner \(\alpha\) et \(\beta\), il nous faut ordonner notre alphabet \(\Sigma\).

Ordre lexicographique

On définit l'ordre lexicographique \(\alpha \leqslant_{lex} \beta\) comme :

Soit \(\alpha = p\alpha'\) et \(\beta = p\beta'\) avec \(p\) étant le plus long préfixe commun de \(\alpha\) et de \(\beta\).

Alors soit

  • \(\alpha' = \epsilon\)
  • \(\alpha' = \alpha'_0\alpha''\) et \(\beta' = \beta'_0\beta''\) et \(\alpha'_0 < \beta'_0\)

\(<\) est l'ordre naturel sur les symboles induits par l'énumération de \(\Sigma\)

\(\leqslant_{lex}\) reste une relation d'ordre mais cette fois est totale.

Exemples

Prenons ici \(\Sigma = \) la table ascii, alors

  • \(a \leqslant_{lex} b\)
  • \(ab \leqslant_{lex} ac\)
  • \(xax \leqslant_{lex} xbb\)
  • \(aa \leqslant_{lex} b\)

Le problème de cet ordre est qu'il existe des mots qui sont précèdes par une infinité d'autres mots. Pour \(\Sigma = \{0, 1\}\), alors \(0 \leqslant_{lex} 00 \leqslant_{lex} 000 \leqslant_{lex} \cdots\) sans jamais atteindre \(1\). Nous avons comme un sentiment que quelque chose manque. C'est pourquoi nous allons introduire un autre ordre, l'ordre alphabétique.

Ordre Alphabétique

On définit l'ordre alphabétique \(\alpha \leqslant_{alp} \beta\) comme :

\(l_{\alpha} < l_{\beta}\) ou \(\alpha \leqslant_{lex} \beta\).

Pour toujours \(\Sigma = {0, 1}\), alors \(0 \leqslant_{alp} 1 \leqslant_{alp} 00 \leqslant_{alp} 01 \leqslant_{alp} 10 \leqslant_{alp}\cdots \)

Nous avons maintenant un ordre bien fondé.
